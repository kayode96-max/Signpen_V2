# **App Name**: SignOff

## Core Features:

- Student Account and Dashboard: Students create and manage their accounts with a personal dashboard to view signatures, customize their page, and generate thank you cards.
- Public Sign-Out Page: Visitors can access a clean and user-friendly signing interface with a digital signature canvas, text tools, and color picker.
- Real-Time Signature Updates: Implement real-time updates using Supabase Realtime to instantly display signatures on the student dashboard and public gallery.
- Customizable Thank-You Message: Allows students to customize a pop-up thank-you message with custom titles, messages, colors, and emojis displayed to signatories after they sign.
- Automated Thank You Card Generation and Design Tool: Generates thank you cards with the signatory’s name, signature preview, message, and timestamp, with options for students to customize backgrounds, add stickers, and edit the message.
- AI-Powered Sentiment Analysis Tool: Leverage generative AI to summarize the sentiments expressed in the collected signatures. Display a summary of overall positive, or specific emotional trends to give an at-a-glance look at how the student is perceived.

## Style Guidelines:

- Primary color: Light Lavender (#E6E6FA) for a soft, celebratory feel.
- Background color: Almost-white Lavender (#F5F5FF) to complement the Lavender while keeping it light and airy.
- Accent color: Soft Purple (#D8BFD8) for CTAs and highlights.
- Body and headline font: 'Alegreya', a humanist serif for an elegant, intellectual, contemporary feel.
- Code font: 'Source Code Pro' for displaying any code snippets, although likely not needed
- Use celebratory and commemorative icons (e.g. graduation caps, stars).
- Subtle animations, such as confetti effects on submission of signatures and thank-you card generation.
